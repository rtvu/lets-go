# Let's Go

The Snippetbox project from [Let's Go](https://lets-go.alexedwards.net/).

# Getting Started

To run locally:

1. Install `go v1.22.3`.

To run with Docker CLI:

1. Run `docker/build`.
2. Run `docker/launch`.

To run with VSCode Dev Containers:

1. Run `docker/build`.
2. In VSCode, run `Reopen in Container`.
